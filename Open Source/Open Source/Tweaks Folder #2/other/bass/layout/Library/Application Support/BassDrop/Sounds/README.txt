/* Information about BassDrop */
Compatible with iOS 10 only.

BassDrop is based on a tweak by the amazing @Ziph0n, previously named Beep Beep for iOS 8/9. 
Changing the iOS charging sound.

Thank you Ziph0n for allowing me to release an updated version.


/* Custom Charging Sounds */
Audio MUST be in a .caf format
Using a custom sound is easy, simply download an audio file with a length of about 1-2s and move it to '/Library/Application Support/BassDrop/Sounds/'
